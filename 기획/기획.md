# 자리톡 둥지트기 도우미

## 배경
* 집을 구하거나 옮길 때 확인해야하는 절차가 많은데, 처음 자취를 시작하는 사람은 뭐부터 어떻게 해야할지 모르는 경우가 많음 
* 집을 구하거나 이사를 해야할 때 확인해봐야할 게 많은데, 빠뜨리기 쉬움
* 기존 자리톡은 매물탐색과 월세납부 및 환급을 도와줬다면, 계약하고 입주하는 과정에서 챙겨야할 것들을 도와주는 도우미를 만들고자함

## 목적
* 새로운 곳에 안전하고 편안하게 둥지를 틀 수 있도록 도와주는 도우미
* 기존 자리톡은 매물탐색과 월세납부 및 환급을 도와줬다면, 계약하고 입주하는 과정에서 챙겨야할 것들을 도와주는 것이 주요 목적

## 기대 효과
* 재방문률 증가. 사용자가 체크리스트를 확인하기 위해 방문하게 되므로 방문 횟수가 잦아질 수 있음
* 재사용 증가. 체크리스트를 통해 확인해야할 서류들을 챙기고, 안전하게 계약했다면 다음번 이사를 할 때도 사용할 확률이 높아짐

## 확장 방향
만약 기대 효과를 충족시킨다면, 전세 사기 시 대응과 같은 방법을 안내하는 방향으로도 확장 가능할 수 있음

## 기능
- 메인 기능: 체크리스트(새집구하기, 이사하기), 안전진단(전세/매매/월세), 용어 사전

## 요구사항
### 공통
- 반응형 디자인 필요. 모바일과 데스크톱 모두 고려
- 프론트엔드 디자인은 front-sample 하위 html 참고

### 체크리스트
- 각 체크리스트는 기획 디렉토리 하위 마크다운 파일 참고
- 체크리스트에서 확인해야하는 사이트는 클릭하면 해당 사이트로 이동 필요
- 체크리스트에서는 사이트마다 확인해야하는 걸 구분하고 구체적으로 명시
- 새집 구하기는 구하는 집의 거주 유형에 따라 체크리스트 구성
- 이사는 상황에 따라 체크리스트 구성. 현재 거주 유형(월세, 전세), 퇴거 방식, 이사할 집 거주 유형에 따라 체크리스트 구성

### 안전진단
- 전세·매매·월세 세 가지 진단 제공 (월세는 간략 버전)
- 기획/전세안전진단-샘플.html, 기획/매매안전진단-샘플.html 디자인 참고
- 데이터 소스, API 연동, 구현 단계는 기획/개발가이드.md 참고

#### 진입 경로
- 메인(상황 선택) 화면 상단에 배너 카드로 노출
  - 문구: "안전하게 집을 구할 수 있도록 도와드릴게요."
  - CTA 버튼: "안전 진단 바로가기" (front-sample/main-front-sample.html의 "내 환급금 확인하기" 스타일 참고)
- 기존 전세가율 계산기 메뉴를 안전진단으로 대체

#### 시세 데이터
- 국토교통부 실거래가 API 자동 조회 (카카오 주소 API → 법정동 코드 → MOLIT API)
- API로 가져온 값은 항목 오른쪽에 `?` 버튼을 제공하여 클릭 시 출처(국토교통부 실거래가) 명시
- 주소 입력은 카카오 주소 검색 팝업 사용

#### 진단 결과 저장
- 비로그인: 저장 없음 (매번 재입력)
- 로그인 시: 마지막 진단 결과를 서버에 저장하여 재방문 시 이어볼 수 있음

#### 전세안전진단
- 사용자 입력: 주소(카카오 팝업), 전세 보증금, 선순위 근저당(등기부등본 직접 확인 후 입력 — 입력 필드 옆에 인터넷등기소 링크 제공), 주택 유형
- 자동 조회: 추정 시세 (국토교통부 실거래가 API)
- 진단 결과:
  - 전세가율 (보증금 ÷ 시세 × 100) + 위험 등급 게이지 (안전 < 70% / 주의 70~80% / 위험 80%+)
  - HUG·HF 전세보증보험 가입 가능 여부 (부채비율 기준 계산)
  - 체크리스트: 투기과열지구 여부, 등기부등본 확인 유도, 건축물대장 확인 유도, 임대인 세금 체납 확인 유도

#### 매매안전진단
- 사용자 입력: 주소(카카오 팝업), 매매 희망가, 주택 유형, 현재 보유 주택 수, 연소득
- 자동 조회: 최근 실거래가 (국토교통부 API), 규제지역 여부
- 진단 결과:
  - LTV 한도 및 최대 대출 가능액 (규제지역·보유 주택 수 기준, `?` 버튼으로 "규제는 자주 바뀔 수 있으므로 참고 지표로만 활용하세요" 안내)
  - DSR (연소득 대비 원리금 상환 비율, 기준 40%)
  - 취득 비용 계산 (취득세, 중개보수, 법무사 비용, 이사비 예상)
  - 체크리스트: 규제지역 여부, 실거주 의무, 재개발/재건축 여부, 최근 실거래가 대비 가격 적정성

#### 월세안전진단 (간략 버전)
- 사용자 입력: 주소, 보증금, 주택 유형
- 진단 결과:
  - 소액보증금 최우선변제 가능 여부 (지역별 기준 적용)
  - 체크리스트: 임대인 체납세금 확인 유도, 등기부등본 확인 유도, 전입신고·확정일자 안내

#### LTV·규제지역 기준 관리
- 어드민에서 수정 가능한 설정값으로 관리 (정부 정책 변경 시 배포 없이 대응)
- 진단 결과 화면에 `?` 버튼: "LTV·DSR 기준은 정부 정책에 따라 자주 바뀔 수 있어요. 본 진단 결과는 참고 지표로만 활용하시고, 정확한 대출 한도는 금융기관에 직접 확인하세요."
