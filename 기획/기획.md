# 자리톡 둥지트기 도우미

## 배경
* 집을 구하거나 옮길 때 확인해야하는 절차가 많은데, 처음 자취를 시작하는 사람은 뭐부터 어떻게 해야할지 모르는 경우가 많음 
* 집을 구하거나 이사를 해야할 때 확인해봐야할 게 많은데, 빠뜨리기 쉬움
* 기존 자리톡은 매물탐색과 월세납부 및 환급을 도와줬다면, 계약하고 입주하는 과정에서 챙겨야할 것들을 도와주는 도우미를 만들고자함

## 목적
* 새로운 곳에 안전하고 편안하게 둥지를 틀 수 있도록 도와주는 도우미
* 기존 자리톡은 매물탐색과 월세납부 및 환급을 도와줬다면, 계약하고 입주하는 과정에서 챙겨야할 것들을 도와주는 것이 주요 목적

## 기대 효과
* 재방문률 증가. 사용자가 체크리스트를 확인하기 위해 방문하게 되므로 방문 횟수가 잦아질 수 있음
* 재사용 증가. 체크리스트를 통해 확인해야할 서류들을 챙기고, 안전하게 계약했다면 다음번 이사를 할 때도 사용할 확률이 높아짐

## 확장 방향
만약 기대 효과를 충족시킨다면, 전세 사기 시 대응과 같은 방법을 안내하는 방향으로도 확장 가능할 수 있음

## 집구하는 절차
1. 매물탐색
2. 계약 전 서류 확인
3. 계약서 작성 & 보증금의 10% 계약금 지금(특약사항 꼭 챙기기)
4. 잔금 지급 및 입주
5. 입주 당일 전입신고, 확정일자 받기
6. 보증보험 가입

## 체크리스트
### 집 알아보기 전
#### 공인중개사 자격확인
- 중개사무소에 등록증 게시 여부 확인
- 중개사 성명 확인
- 국가공간정보포털의 브이월드 부동산중개업 조회 등에서 등록 여부 확인

특히 계약서에 서명하는 사람이 실제 개업공인중개사인지 확인하는 것이 중요합니다.

### 마음에 드는 집을 찾았을 때
#### 대출 확인
- 최대 대출 가능 금액
- 예상 금리
- 소득 기준 충족 여부
- 전세대출 가능 여부

### 계약 전
#### 등기부등본 ✅ 제일 중요
- 확인방법: 대법원 인터넷등기소 (iros.go.kr) / 무인발급기
- 비용: 700원
- 확인할 것:
  - 근저당 설정 금액 (집주인 빚)
  - 가압류/압류 여부
  - 실제 소유자가 계약자와 같은지
  - 소유권 이전 이력 (최근에 자주 바뀌었으면 주의)
#### 건축물대장
- 확인방법: 정부24 (gov.kr) / 무인발급기
- 비용: 무료
- 확인할 것:
  - 위반건축물 여부
  - 주택 용도가 맞는지 (근린생활시설인데 주거용으로 쓰는 경우 주의)
  - 불법 증축 여부
#### 토지대장 / 토지이용계획확인서
- 확인방법: 정부24
- 비용: 무료
- 확인할 것:
  - 개발제한구역 여부
  - 재개발/재건축 구역 여부 (계약 후 갑자기 철거되는 경우 방지)
#### 계약서 확인
- 필요한 이유: 계약 당일 모호한 문구가 있거나 말이 달라질 수 있기 때문에, 확인이 가능할 경우 미리 확인하면 좋습니다.
- 확인방법: 공인중개사에게 요청. 계약서 전체를 받을 수 없다면, 특약 문구를 미리 받을 수 있는지 확인
- 확인할 것
  - 특약 가능 여부 확인(대출할 경우, "전세자금대출이 승인되지 않을 경우 계약금을 반환하고 계약을 해제한다." 는 항목 확인 필요)
#### 임대인 세금 체납 여부 확인
- 필요한 이유: 집이 경매로 넘어갈 경우 일부 국세·지방세는 임차보증금보다 먼저 변제될 수 있습니다.
- 확인방법:
  - 임대인 동의 후 세무서 확인
  - 임차인 정보제공 제도를 통한 확인
- 확인할 것
  - 국세 체납 여부
  - 지방세 체납 여부

### 계약 당일
#### 계약서 임대인
- 확인할 것: 
  - 계약 전 확인한 등기부등본 소유자가 계약서 임대인과 동일한지 확인

#### 계약 몰취 조건
- 확인할 것:
  - 계약 해제 시 계약금이 어떻게 되는지

#### 대출 특약 확인
- 확인할 것:
  - "전세자금대출이 승인되지 않을 경우 계약금을 반환하고 계약을 해제한다." 는 항목 확인 필요
  - "대출이 불가할 경우 협의하여 처리한다." 와 같이 모호한 문구가 있을 경우, 명확히 해야함. 추가 조건을 펜으로 적을 경우 추후 분쟁 여지가 있기 때문에, 수정된 부분 옆에 임대인과 임차인의 서명 또는 간인을 해둬야합니다.

#### 전세보증보험 특약 확인
- 확인할 것:
  - "임대인은 임차인의 전세보증보험 가입에 협조한다." 는 항목 확인 필요

### 입주 당일
- 확정일자
  - 확인방법: 주민센터 / 등기소 / 인터넷 등기소
  - 비용: 600원
  - 중요성: 집이 경매로 넘어갔을 때 보증금 우선 변제 순위 확보 (전입신고랑 같은 날 받는 게 좋음)
- 전입신고
  - 확인방법: 주민센터 / 정부24
  - 비용: 무료
  - 중요성: 대항력 발생 → 집이 팔려도 새 주인한테 계속 거주 가능 (입주 당일 바로 해야 함)

### 입주 이후
- 보증보험 가입
