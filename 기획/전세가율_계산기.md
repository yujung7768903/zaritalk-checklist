# 전세가율 계산기

## 목적
- 계약하려는 집이 깡통전세 위험이 있는지 한눈에 판단
- 매매가·전세가를 각각 여러 사이트에서 확인해야 하는 번거로움을 하나의 화면에서 해결

## 진입 방법
- 새집 구하기 체크리스트 > "전세가율 확인" 항목 내 바로가기 버튼
- 메인 화면 서브 기능 메뉴에서 직접 진입

---

## 입력값

| 항목 | 설명 | 단위 |
|------|------|------|
| 매매가 | 해당 집의 최근 매매가 | 만원 |
| 전세가 | 계약 예정인 전세 보증금 | 만원 |

- 두 값 모두 사용자가 직접 입력 (외부 API 조회 없음)
- 입력창 하단에 참고 사이트 링크 제공 (국토교통부 실거래가, KB부동산, 네이버부동산)

---

## 계산식

```
전세가율(%) = 전세가 ÷ 매매가 × 100
안전 여유금  = 매매가 - 전세가
```

---

## 출력값

### 전세가율 위험도

| 전세가율 | 등급 | 색상 |
|----------|------|------|
| 70% 미만 | 안전 | 초록 |
| 70% 이상 ~ 80% 미만 | 보통 | 노랑 |
| 80% 이상 ~ 90% 미만 | 주의 | 주황 |
| 90% 이상 | 위험 | 빨강 |

### 부가 정보
- **안전 여유금**: `매매가 - 전세가` (경매 낙찰 시 보증금 회수 가능한 여유 금액)
- **위험도 설명 문구**: 등급에 따른 한 줄 안내 메시지

| 등급 | 안내 문구 |
|------|-----------|
| 안전 | 전세가율이 낮아 비교적 안전한 편입니다. |
| 보통 | 전세가율이 다소 높습니다. 등기부등본과 선순위 채권을 꼭 확인하세요. |
| 주의 | 전세가율 80% 이상입니다. 전세보증보험 가입을 반드시 확인하세요. |
| 위험 | 깡통전세 위험이 높습니다. 계약 전 전문가 상담을 권장합니다. |

---

## UX 상세

- 숫자 입력 시 콤마 자동 삽입 (예: 30,000만원)
- 매매가·전세가 두 값 모두 입력되면 자동으로 결과 표시
- 결과 하단에 "매매가 확인하러 가기" 외부 링크 버튼 제공
- 초기화 버튼으로 다시 계산 가능

---

## 계산 예시

| 매매가 | 전세가 | 전세가율 | 안전 여유금 | 등급 |
|--------|--------|----------|-------------|------|
| 3억원 | 1억 8천 | 60% | 1억 2천 | 안전 |
| 3억원 | 2억 4천 | 80% | 6천만원 | 주의 |
| 3억원 | 2억 7천 | 90% | 3천만원 | 위험 |

---

## 참고 사항
- 전세가율은 지역·시세에 따라 다르므로 최신 실거래가 기준으로 입력
- 선순위 근저당이 있는 경우 `(근저당 채권최고액 + 전세가) ÷ 매매가`로 실질 위험도를 별도 계산 권장 → 향후 확장 기능으로 고려
