# 서비스 UX 흐름

## 플랫폼 및 기술 방향
- **플랫폼**: 모바일 웹 우선 (기존 자리톡 tenant.zaritalk.com과 동일)
- **디자인 시스템**: 기존 자리톡 UI 컴포넌트 재사용 (흰 카드, 파란 포인트 컬러 #2C7FFF)
- **상태 저장**: 비로그인 시 localStorage, 로그인 시 서버 동기화

---

## 앱 내 진입점

기존 자리톡 메인 탭 바에 **"이사준비"** 탭 또는 메인 홈 화면 내 카드 배너로 진입.

```
메인 홈
  └─ "둥지트기 도우미" 배너 or 탭 메뉴
        └─ 상황 선택 화면
```

---

## 상황 선택 화면

진입 시 사용자가 자신의 상황을 선택.

```
어떤 상황인가요?

[ 새집 구하기      ]  ← 처음 자취 or 기존 집 없이 새로 구하는 경우
[ 전세 이사하기    ]  ← 현재 전세 살고 있고 새 전세로 이사
[ 월세 이사하기    ]  ← 현재 월세 살고 있고 새 곳으로 이사
```

선택 후 해당 체크리스트 화면으로 이동.

---

## 체크리스트 화면 구조

### 전체 진행률
- 상단에 전체 항목 대비 완료 항목 수 표시
- 예: `12 / 28 완료`  + 프로그레스 바

### 단계별 섹션
체크리스트는 단계(집 알아보기 전 / 계약 전 / 계약 당일 / 잔금 전 / 입주 당일 / 입주 이후)로 구분.
각 단계는 접기/펼치기 가능.

```
[▼ 계약 전]  3/6 완료
  ☑ 등기부등본 확인         → 인터넷등기소 바로가기 [↗]
  ☑ 건축물대장 확인         → 정부24 바로가기 [↗]
  ☐ 임대인 세금 체납 확인
  ☐ 전세가율 확인          → 전세가율 계산기 [→]
  ☐ 전세보증보험 가능 여부   → HUG 바로가기 [↗]
  ☐ 계약서 사전 확인
```

### 체크 항목 상세
각 항목을 탭하면 하단 바텀시트로 상세 정보 표시:
- 필요한 이유
- 확인방법
- 비용
- 확인할 것 (서브 체크박스)
- 관련 외부 사이트 링크 버튼

---

## 용어 연동 방식

체크리스트 및 상세 화면에 등장하는 용어(예: 근저당, 전세가율, 확정일자 등)는 **밑줄 표시** 후 탭 시 바텀시트로 용어 설명 팝업 노출.

```
"근저당 확인" 항목에서 "근저당" 단어 탭
  → 바텀시트 팝업
    ┌──────────────────┐
    │ 근저당            │
    │ 은행이 설정한 담보권 │
    │ 쉽게 말하면 집주인의 │
    │ 빚                │
    └──────────────────┘
```

용어사전 전체 보기 페이지도 별도 제공.

## 체크리스트 상태 저장

### 비로그인
- localStorage에 진행 상태 저장
- 앱/브라우저 재방문 시 이어서 진행 가능
- 기기 변경 시 초기화됨

### 로그인
- 서버에 진행 상태 저장
- 여러 기기에서 이어서 진행 가능
- 로그인 유도 배너: "로그인하면 진행 상태가 저장돼요"

### 복수 체크리스트
현재 버전에서는 하나의 체크리스트만 진행.
재시작 버튼으로 초기화 가능 (재시작 전 확인 다이얼로그 표시).

## 서브 기능 메뉴 구성

체크리스트 외 별도 제공하는 도구:

| 메뉴 | 설명 |
|------|------|
| 전세가율 계산기 | 매매가·전세가 입력 후 위험도 계산 |
| 용어사전 | 부동산·계약 관련 용어 검색 및 설명 |

## 화면 목록

| 화면 | 설명 |
|------|------|
| 상황 선택 | 새집 구하기 / 전세이사 / 월세이사 선택 |
| 체크리스트 메인 | 전체 진행률 + 단계별 항목 목록 |
| 항목 상세 바텀시트 | 항목 설명, 외부 링크, 서브 체크박스 |
| 용어 설명 바텀시트 | 용어 탭 시 노출되는 팝업 |
| 용어사전 | 전체 용어 목록 및 검색 |
| 전세가율 계산기 | 매매가·전세가 입력 및 결과 표시 |
